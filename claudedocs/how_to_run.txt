There are two entry points in this repo:

1) FastAPI backend (templates + static)
   a) Optional: download an embedding model locally
      - python scripts/download_model.py -m sentence-transformers/all-MiniLM-L6-v2 \
        -o ./models/all-MiniLM-L6-v2 --method st
      - export EMBEDDING_MODEL=$(pwd)/models/all-MiniLM-L6-v2

      Or use a HF Hub ID directly (requires network on first run):
      - export EMBEDDING_MODEL=all-MiniLM-L6-v2
      - Optional fallback: export EMBEDDING_FALLBACK=jinaai/jina-embeddings-v2-base-zh

   b) Run with Uvicorn:
      - uvicorn app_fastapi_ollama:app --reload
      - Then open http://localhost:8000/

   Note: Running this file with Streamlit will fail or render nothing,
   because itâ€™s not a Streamlit app. If you previously saw
   "RuntimeError: Directory 'static' does not exist", that is fixed by
   the `static/` and `templates/` folders already included.

2) Streamlit UI example
   - Original app: streamlit run app.py
   - Custom component demo: streamlit run streamlit_components/demo_app.py
